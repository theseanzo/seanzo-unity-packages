# Changelog

All notable changes to this package are documented here. The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [0.1.0] - Unreleased

### Added
- Grid Paint scene-view tool: brush, box fill, flood fill, and erase across floor (XZ) and wall (XY/YZ) paint planes, with placement previews, layers, and full Undo support.
- Kit Palette window: drag-and-drop prefab registration, footprint detection, authored facing/up axes with auto-detect, footprint overrides, and per-entry rule tiles.
- Rule Kit Tile asset: 3x3 in-plane neighbor matching with rotation-aware rule reuse and automatic neighbor re-resolution on paint and erase.
- Pivot editor: prefab-stage tool with placement preview frames, floor/wall fit snaps, and per-axis pivot snapping.
- Kit prep utilities: Create Kit Prefabs From Selection and Blender Z-up correction menus.
- Edit-mode test suite under Tests/Editor.
